# Capoo-Survival.

一个以猫猫虫为主题的幸存者 like 游戏，使用 SFML 作为游戏的多媒体库。

## Getting Started

Just download the newest "Capoo-Survival" in the [Releases](https://github.com/tLLWtG/Capoo-Survival/releases) of our project, and enjoy it.
